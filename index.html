Facebookless

The chrome extension that help people be more productive and use facebook less

Download this project as a .zip file Download this project as a tar.gz file

Help you become more productive by making Facebook less addicting.

Let's be more productive by using Facebook less.

After a week not too productive, I decided to make this extension to help me focus on working instead of endlessly scrolling through my Facebook feed.

It will show you motivating reasons why you should close Facebook tab and start working. You can also set a time range to hide news-feed, you can show this news-feed up by confirm 2 questions: "Where is that wasting my time news-feed?" and "Show me the news-feed that wasting my time!"

Features

  1. Show you why you should close facebook tab and start working
  2. Hide facebook newsfeed and require you to click 2 links to show newsfeed (enable from 9AM to 6PM by default)
